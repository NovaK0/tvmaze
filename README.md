# TVMaze

TVMaze is a web application built using React that fetches and displays information about movies and TV shows from the TVMaze API. The app allows users to search for their favorite movies or shows and view detailed information about them.

## Features

- **Search Functionality:**
  - Search for movies and TV shows by title.
- **Detailed Information:**
  - View detailed information about each movie or show, including cast, genres, summary, and air date.
- **Responsive Design:**
  - Fully responsive design, optimized for both desktop and mobile devices.
- **API Integration:**
  - Fetches real-time data from the TVMaze API.
